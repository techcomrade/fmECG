# bluetooth_ecg

`
flutter build apk --no-shrink --no-tree-shake-icons
`

**NOTE**
**Java version**: 17
Change **JAVA_HOME** in `gradle.properties` (Check your Java path in Android Studio -> File -> Project Struture -> SDKs -> Choose SDK Java 17 -> Copy path)
